# Terminal Portfolio

This project is a portfolio website that functions like a shell terminal. Users can type custom commands to view information about work experience, projects, and leadership experience.

## Features

- Interactive terminal interface
- Custom commands to display various information
- Responsive design

## Installation

To get started with this project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd terminal-portfolio
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To run the application, use the following command:
```
npm start
```
This will start the development server and open the application in your default web browser.

## Commands

Here are some of the commands you can use in the terminal:

- `experience`: Displays work experience.
- `projects`: Lists projects.
- `leadership`: Shows leadership experience.
- `about`: Provides a brief introduction.
